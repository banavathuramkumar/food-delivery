# 🍔 Cravora – Full Stack Food Delivery Web App

🚀 A complete **MERN Stack Food Delivery Application** with separate **User** and **Admin** panels, real-time order flow, and **Stripe payment integration**.

✨ You can **try the live demo here:** 

**User Page:** https://food-del-frontend-4q0b.onrender.com/

**Admin Page:** https://food-del-admin-oz2m.onrender.com/


---

## 📌 Project Overview

**Cravora** is a full-stack food ordering platform inspired by real-world apps like Zomato and Swiggy. It allows users to browse food items, manage carts, and place orders, while admins can manage products and track orders through a dedicated dashboard. 

---

## ✨ Features

👨‍🍳 User Side (Frontend)

- 🔐 User Authentication (Sign Up / Login)

- 🍽️ Browse food menu with images & prices

- 🛒 Add to Cart & Update Quantity

- 💰 Cart Calculation (Subtotal + Delivery Fee)

- 🎟️ Promo Code Input

- 💳 Payment Options:

- Cash on Delivery (COD)

- Stripe (Credit/Debit Card)

- 📦 Order Placement with Delivery Details

- ⚡ Smooth & Responsive UI

---


🧑‍💼 Admin Panel

- 📊 Separate Admin Dashboard

- ➕ Add New Food Items

- ✏️ Update Existing Items

- ❌ Delete Items

- 📦 Manage Orders

- 🔄 Real-time Data Sync with Frontend

## ⚙️ System Architecture


```
    👤 User (React Frontend)
                 │
                 │  API Requests
                 ▼
 ⚙️ Node.js + Express Backend
                 │
                 │  CRUD Operations
                 ▼
       🗄️ MongoDB Database

```
```
    🧑‍💼 Admin Panel (React)
                 │
                 │  Same REST APIs
                 ▼
 ⚙️ Node.js + Express Backend
 ```
